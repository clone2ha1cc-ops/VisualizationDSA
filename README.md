> **"Biến sự trừu tượng thành hình ảnh - Nền tảng EdTech tối thượng cho Khoa học Máy tính."**

## 🚀 Tính năng Khác biệt (Unique Selling Propositions - USP)

Nếu Phase 1 tập trung tái tạo lại nền tảng vẽ thuật toán cổ điển, thì **Phase 2 chính là linh hồn và lợi thế cạnh tranh độc tôn** của dự án này:

1. **Trực quan hóa Khái niệm Trừu tượng (Abstract Concepts Visualization):** Thay vì chỉ vẽ Mảng (Array) hay Cây (Tree), hệ thống có khả năng mô phỏng:
   - Các tính chất Lập trình hướng đối tượng (**OOP**): Kế thừa, Đa hình, Đóng gói.
   - Nguyên tắc thiết kế **SOLID**: Trực quan hóa Single Responsibility, Dependency Inversion bằng các "nhà máy" và "ống dẫn dữ liệu".
   - **Dependency Injection (DI) Container**: Xem cách các Services được khởi tạo và tiêm (inject) vào nhau ở Runtime.
2. **E-Lecture Mode (Bài giảng kịch bản):** Dẫn dắt người dùng từng bước thông qua các Tutorial tương tác thay vì ném cho họ một ma trận code khó hiểu.
3. **Interactive Playground:** Cung cấp Canvas để người dùng tự do vẽ Node, Edge, xây dựng đồ thị của riêng mình để thuật toán chạy trên đó.
4. **Data-driven Animation Engine:** Backend C# sinh ra dữ liệu, Frontend Vue chỉ đóng vai trò như một "Video Player" để phát (Playback) dữ liệu, đảm bảo không bao giờ giật lag hay sai lệch logic.

## 🛠️ Công nghệ cốt lõi (Tech Stack)

### Backend (The Brain)
- **Framework:** .NET 8/9, C#, ASP.NET Core Web API.
- **Architecture:** Clean Architecture, Strategy Pattern cho từng module thuật toán.
- **Responsibility:** Chạy thuật toán trong bộ nhớ, bắt giữ (capture) sự thay đổi trạng thái (State Frames) và sinh mảng JSON tối ưu hóa (GZIP).

### Frontend (The Canvas)
- **Framework:** Vue 3 (Composition API), TypeScript, Vite.
- **State Management:** Pinia (Tách biệt AnimationStore, InputStore, LectureStore).
- **Graphics Engine:** Native HTML5 Canvas 2D / SVG hoặc WebGL (Three.js/Pixi.js) tùy thuộc vào độ nặng của thuật toán.
- **Styling:** TailwindCSS, SCSS (Dark/Light mode native).
