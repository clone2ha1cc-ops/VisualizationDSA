# 🔮 Định Hướng Kế Hoạch Tương Lai - Future Extension Blueprint

Tài liệu này vạch ra các hướng đi nghiên cứu phát triển các tính năng đột phá trong tương lai cho nền tảng **VisualizationDSA** nhằm đón đầu các xu thế công nghệ EdTech mới.

---

## 1. Tính Năng 1: Trợ Lý Ảo Phân Tích Code Bằng Generative AI (Gemini AI Debugger)
*   **Mô tả:** Tích hợp trực tiếp trợ lý AI thông minh (mô hình Gemini 1.5 Pro) vào Monaco Editor.
*   **Hành vi trực quan:** 
    *   Học viên viết code tùy biến, AI tự động quét AST để phân tích độ phức tạp thời gian thực tế ($O(N \log N)$ hay $O(N^2)$).
    *   AI đưa ra các gợi ý tái cấu trúc mã (Refactoring tips) và giải thích cặn kẽ các dòng code vi phạm quy tắc tối ưu hóa hoặc SOLID ngay trong khung chat Glassmorphism Neon mờ ảo.

---

## 2. Tính Năng 2: Đấu Trường Lập Trình Giải Thuật Thời Gian Thực (Multiplayer Coding Duels)
*   **Mô tả:** Chế độ đấu đối kháng học tập trực tuyến kết nối thông qua giao thức WebSockets trễ cực thấp.
*   **Hành vi trực quan:**
    *   Ghép cặp 2 học viên trong phòng thi đấu giải câu đố DSA (ví dụ: Viết code xoay cây AVL).
    *   Màn hình chia đôi Glassmorphism. Mỗi bên hiển thị Monaco Editor và Canvas trực quan hóa của đối thủ thời gian thực.
    *   Bên nào tối ưu hóa code chạy đúng, cân bằng cây AVL nhanh nhất sẽ giành chiến thắng, cướp cúp và thăng hạng huy hiệu Neon rực rỡ.

---

## 3. Tính Năng 3: Trình Xuất Bản Phim Hoạt Ảnh DSA Độ Phân Giải Cao (HD MP4/WebM Animation Exporter)
*   **Mô tả:** Cho phép giáo viên EdTech xuất bản trực tiếp các bài trực quan giải thuật thành video.
*   **Hành vi trực quan:**
    *   Kéo scrubber chọn khoảng thời gian VCR playback mong muốn.
    *   Nhấp phím "Export HD Video", hệ thống sử dụng thư viện đệm `MediaRecorder API` chụp từng khung hình Canvas 2D 60 FPS chất lượng cao, ghép thành tệp tin video MP4/WebM sắc nét để đăng lên Youtube, Tiktok hoặc bài giảng khóa học.
